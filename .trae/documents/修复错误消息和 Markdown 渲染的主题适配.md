## 问题分析

当前的 SystemToast 设计：
1. 使用 QFrame 作为容器
2. 背景色使用 error_bg（浅红色）
3. 边框和背景色相同
4. 布局是简单的水平居中
5. 左右边距较大（40px）

用户反馈 UI 太丑，需要重新设计。

## 改进方案

1. **更紧凑的设计**：
   - 减少边距
   - 使用更现代的卡片样式

2. **更好的视觉层次**：
   - 图标和文字更好的对齐
   - 添加左侧强调色边框

3. **改进颜色方案**：
   - 背景使用更 subtle 的颜色
   - 添加左侧强调边框

4. **更好的布局**：
   - 图标和文字垂直居中对齐
   - 文字左对齐而非居中

重新设计 SystemToast 类：
1. 使用更紧凑的边距
